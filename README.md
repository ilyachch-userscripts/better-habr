# Better Habr

Tampermonkey app

## Installation

1. Install [Tampermonkey](https://www.tampermonkey.net/).
2. [Click here to install](https://github.com/ilyachch-userscripts/better-habr/releases/latest/download/better-habr.user.js).

## Development

```bash
# Install dependencies
npm install

# Start dev server (with HMR)
npm run dev

# Build for production
npm run build
```
